A GUI application was created using MATLAB's AppDesigner and Deploytool. The program is based on the image encrytion code presented here, which encrypts an image with a unique binary key. The encrypted image and unique key can be sent to a recipient who has the program installed, where the decryption process can take place to reconstruct the image.
